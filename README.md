# Parcial 1 - Programa de Préstamos

## Descripción
Programa en C++ que permite evaluar y calcular un préstamo según:
- Tipo de cliente
- Monto solicitado
- Plazo
- Si tiene codeudor

El programa valida las condiciones y muestra si el préstamo es aprobado o rechazado, además de calcular los intereses y el total a pagar.

---

## Tipos de cliente
1. Estudiante  
2. Empleado  
3. Pensionado  
4. Desempleado  

---

## Fórmula utilizada
Interés:
interes = monto * tasa * plazo  

Total:
total = monto + interes
